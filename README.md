# Frontend Application

## Overview

React.js frontend for Ismail Sanouni Portfolio website.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000/api
```

## Running

Development:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Tech Stack

- React 18+
- Vite
- TailwindCSS
- React Router 6
- Framer Motion
- Axios
